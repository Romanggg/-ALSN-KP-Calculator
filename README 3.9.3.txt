Калькулятор КП АЛСН Ver 3.9.3
📋 Обновление от 02.07.2026
🎯 Краткое описание изменений
Версия 3.9.3 — это значительное обновление, направленное на полную автоматизацию валидации всех параметров в журнале Д.4 и единообразную цветовую индикацию в результатах расчёта. Теперь все проверяемые параметры имеют наглядную визуальную обратную связь: зелёный цвет — норма, красный — отклонение, с автоматическим формированием замечаний.

📊 Полный список изменений
1. Журнал Д.4 — полная автоматическая валидация
1.1. Валидация токов на всех частотах (25, 50, 75, 175 Гц)
До: токи выводились как текст без проверки.

После: каждый ток проверяется на соответствие нормативному диапазону.

Цветовая индикация: зелёный (d4-valid) — в норме, красный (d4-invalid) — вне нормы.

Автоматическое замечание: при превышении нормы добавляется текст «⚠ Ток вне нормы!».

1.2. Валидация коэффициента экранирования (kэЛ и kэП)
До: коэффициенты выводились одной строкой kэЛ = X.XXX, kэП = X.XXX без проверки.

После: для каждого коэффициента добавлена отдельная строка с проверкой ≥ 1.0.

Цветовая индикация: зелёный — kэ ≥ 1.0, красный — kэ < 1.0.

Автоматическое замечание: при kэ < 1.0 добавляется текст «⚠ kэЛ < 1.0» или «⚠ kэП < 1.0».

1.3. Валидация расхождения коэффициентов экранирования
До: расхождение выводилось как текст без проверки.

После: добавлена автоматическая проверка расхождения ≤ 5%.

Цветовая индикация: зелёный — расхождение ≤ 5%, красный — расхождение > 5%.

Автоматическое замечание: при превышении добавляется текст «⚠ Превышение! Требуется регулировка подвески катушек».

2. Результаты расчёта — единообразная цветовая индикация
2.1. Измеренное сопротивление
До: ✓ 1.100 Ом ≤ 1,2 Ом (без цвета)

После: 🟢 ✓ 1.100 Ом ≤ 1,2 Ом или 🔴 ⚠ 1.350 Ом > 1,2 Ом

2.2. Сопротивление изоляции
До: ✓ 100 кОм ≥ 50 кОм (без цвета)

После: 🟢 ✓ 100 кОм ≥ 50 кОм или 🔴 ⚠ 40 кОм < 50 кОм

2.3. Высота шлейфа
До: ✓ Отклонение 3.0 мм в пределах ±10 мм (без цвета)

После: 🟢 ✓ Отклонение 3.0 мм в пределах ±10 мм или 🔴 ⚠ Отклонение 15.0 мм > ±10 мм

2.4. Расстояние до изолирующего стыка
До: ✓ 5.0 м в диапазоне 0,2–30 м (без цвета)

После: 🟢 ✓ 5.0 м в диапазоне 0,2–30 м или 🔴 ⚠ 35.0 м вне диапазона 0,2–30 м

3. Обновление функции checkValueAgainstNorm()
Добавлена поддержка операторов «≤» и «<» для корректной обработки норм вида «≤ 5 %» и «< 50».

Поддерживаемые форматы норм:

≥ число — больше или равно

> число — больше

≤ число — меньше или равно (НОВОЕ)

< число — меньше (НОВОЕ)

число–число — интервал

— — норма не задана (проверка не выполняется)

4. Исправление формата норм для токов в журнале Д.4
До: '1,40–1,47 А' (с единицей измерения) — функция checkValueAgainstNorm() не могла обработать.

После: '1,40–1,47' (без единицы) — функция корректно проверяет диапазон.
