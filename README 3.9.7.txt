Изменения для GitHub (версия 3.9.7 – мобильная адаптация)
Версия 3.9.7 (мобильная адаптация)
📱 Основные изменения
Добавлена полноценная адаптивная вёрстка для мобильных устройств и планшетов.

Оптимизирована производительность на слабых мобильных устройствах (debounce, отключение анимации, уменьшение перерисовок).

📋 Детальный список изменений
1. Viewport и базовая адаптация
Добавлен мета-тег:
<meta name="viewport" content="width=device-width, initial-scale=1.0">
для корректного масштабирования на мобильных экранах.

2. Новые медиа-запросы
@media (max-width: 768px) – для планшетов и телефонов в портретной ориентации:

Все основные сетки (.grid, .input-grid, .add-grid) переключаются в одну колонку.

Блок «Технические параметры» (второй блок) также переводится в одну колонку.

Блок «Параметры приёмных катушек» отображается в две колонки (для более компактного размещения).

Уменьшены отступы, размеры шрифтов и кнопок.

Отключена анимация fadeInUp для результатов.

@media (max-width: 480px) – для маленьких телефонов:

Блок «Параметры приёмных катушек» переключается в одну колонку.

Дополнительно уменьшены отступы и размеры элементов.

@media (max-width: 900px) and (orientation: landscape) – для горизонтальной ориентации телефонов:

Блок «Конфигурация системы» – 3 колонки (вместо 5).

Блок «Технические параметры» – 2 колонки.

Блок «Параметры ПК» – 2 колонки.

Дополнительные параметры – 2 колонки.

Группы ЭДС (emf-freq-group) – возвращаются к 3 колонкам.

Модальные окна расширяются до 90% ширины экрана.

3. Оптимизация производительности
Debounce для сохранения настроек – функция saveSettings() обёрнута в debounce с задержкой 500 мс, чтобы уменьшить частоту записи в localStorage при активном вводе.

Отключена анимация на мобильных устройствах (.result-card не анимируется, чтобы избежать тормозов).

Уменьшены размеры полей ввода и кнопок на мобильных для ускорения рендеринга.

4. Улучшения интерфейса
Уменьшены отступы (padding) внутри блоков и карточек.

Размер шрифта для меток и полей ввода адаптирован под маленькие экраны.

Модальные окна теперь занимают 95% ширины на мобильных (вместо 90%).

Исправлено отображение таблиц в модальных окнах (добавлен overflow-x: auto для горизонтальной прокрутки).

5. Исправления
Устранена проблема с перекрытием элементов на узких экранах.

Исправлено отображение поля «Сопротивление изоляции» в блоке «Технические параметры» – теперь оно занимает стандартную ширину колонки.

🖥️ Совместимость
Калькулятор оптимизирован для работы на всех современных браузерах (Chrome, Firefox, Edge, Safari) как на десктопе, так и на мобильных устройствах (Android, iOS).

В горизонтальной ориентации телефонов интерфейс автоматически перестраивается для удобного использования.
